# TypeFlow

A Figma plugin that generates typography systems, tokens, text styles, and developer-ready outputs.

## Features

- Generate typography scale
- Support mobile / tablet / desktop modes
- Create text styles and variables
- Bind styles to variables
- Export JSON / CSS
- Import JSON
- Save and load presets

## Installation

1. Download this repository
2. Open Figma
3. Go to Plugins → Development → Import plugin from manifest
4. Select `manifest.json`
5. Run TypeFlow

## Why TypeFlow

Typography systems are often fragmented across styles, tokens, and dev outputs.

TypeFlow connects them into a single workflow:
define → generate → bind → export → reuse
