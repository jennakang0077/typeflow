# TypeFlow Design System

## Overview
A clean, modern design system engine UI for designers and design engineers.

Focused on clarity, structure, and productivity rather than decoration.

The interface should feel like a hybrid between a Figma plugin and a professional developer tool.

---

## Layout

- Three-panel layout:
  - Left: Token & Style Configuration
  - Center: Live Typography Preview
  - Right: Generated Output (JSON / CSS / Variables)

- Top bar for project name and actions
- Sticky action bar for Apply / Export / Import

- Each panel scrolls independently
- Dense but readable layout

---

## Colors

- Primary: #5B8CFF
- Surface: #FFFFFF
- Surface-alt: #F7F8FA
- Border: #E5E7EB
- Text-primary: #111827
- Text-secondary: #6B7280
- Accent: #7C3AED
- Error: #EF4444

Use color sparingly. Prioritize clarity over decoration.

---

## Typography

- Font: Inter

- Heading:
  - SemiBold
  - Clear hierarchy (H1–H4)

- Body:
  - Regular
  - 14–16px

- Label:
  - Medium
  - 12px

Typography should feel precise and readable.

---

## Spacing System

Base unit: 4px

Common spacing:
- 4 / 8 / 12 / 16 / 24 / 32

Maintain consistent spacing rhythm.

---

## Components

### Panels
- Light background (Surface-alt)
- Subtle borders
- No heavy shadows

### Cards / Sections
- Radius: 8px
- Light border
- Padding: 16–20px

### Buttons
- Primary: filled
- Secondary: outline or subtle
- Radius: 6–8px

### Inputs
- Minimal styling
- 1px border
- Clear focus state

---

## Interaction

- Fast and responsive
- Minimal animation
- Subtle hover and focus states

---

## Do’s

- Keep layout structured and predictable
- Maintain strong hierarchy
- Make scanning easy
- Use spacing for clarity

---

## Don’ts

- No excessive shadows
- No heavy decoration
- No inconsistent styles
- Avoid marketing-style UI

---

## Visual Tone

- Professional
- Minimal
- Tool-like
- Efficient

---

## Inspiration

- Stripe (spacing & structure)
- Figma (panel layout)
- Linear (minimal aesthetic)
